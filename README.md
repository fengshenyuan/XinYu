# XinYu
XinYu -《新语》目标为尝试整理编辑自2000年以来中文文化圈、中文互联网上出现的汉语新词汇、新俗语、或本意被翻新后再次大规模流行的“成语”，流行词，汇集成典，以供读者参考，或做汉语言文字学研究材料用。比如人从众𠈌，喜大普奔，十动然拒，宏彥获水等新式“成语”，内卷，破防，灌水，真香，吃瓜群众，图样图森破，YYDS，凡尔赛，新质生产力等汉语新词汇。
